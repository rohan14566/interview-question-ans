Q.(5)difference between inline and block

inline:

  • inline element allows other inline element to sit behind
  • take space as much as needed
  • eg-(a),(img) (span) tags are inline element

block:

  • block element does not allow other element to sit behind
  • full width is occupied
  • its start with new line
  • eg-(div),(p),(h1),(footer) tags are block element
